SwiftSync — Tecnologias em uso (Março 2026)

Este documento lista as tecnologias identificadas no projeto e agrupa-as por categoria.
Fontes usadas: package.json, schema Prisma, configs (Next/ESLint/PostCSS/TS), e código de integrações.

=====================================================================
1) LINGUAGENS, RUNTIME E PLATAFORMA
=====================================================================
- TypeScript: linguagem principal da aplicação (frontend, backend e scripts TS).
- JavaScript: usado em scripts utilitários e ficheiros de configuração.
- Node.js: runtime para execução do servidor Next.js, scripts e tooling.
- React 19.2.0: biblioteca base para construção da interface.
- React DOM 19.2.0: renderização React no browser.

=====================================================================
2) FRAMEWORKS PRINCIPAIS
=====================================================================
- Next.js 16.0.7: framework full-stack (App Router, SSR, rotas API, middleware).
- Prisma 7.3.0 / 7.4.0: ORM para modelação de dados, queries tipadas e migrações.

=====================================================================
3) BASE DE DADOS E ACESSO A DADOS
=====================================================================
- PostgreSQL: base de dados principal (provider do Prisma: postgresql).
- pg 8.18.0: driver PostgreSQL para Node.js.
- @prisma/client 7.3.0: client gerado pelo Prisma para acesso tipado à BD.
- @prisma/adapter-pg 7.4.0: adapter Prisma para PostgreSQL.

=====================================================================
4) UI, DESIGN SYSTEM E EXPERIÊNCIA
=====================================================================
- Tailwind CSS 4: framework utilitário de estilos.
- @tailwindcss/postcss 4: plugin PostCSS para compilar Tailwind.
- shadcn/ui (configurado por components.json): base de componentes reutilizáveis.
- Radix UI (pacotes @radix-ui/*): primitives acessíveis para componentes de UI.
  - react-avatar: avatars.
  - react-checkbox: checkboxes.
  - react-collapsible: áreas expansíveis.
  - react-dialog: modais/dialogs.
  - react-dropdown-menu: menus dropdown.
  - react-label: labels acessíveis.
  - react-popover: popovers.
  - react-select: selects custom.
  - react-separator: separadores visuais.
  - react-slot: composição de componentes.
  - react-switch: toggles.
  - react-tabs: tabs.
  - react-toggle: botão toggle.
  - react-toggle-group: grupos de toggles.
  - react-tooltip: tooltips.
- lucide-react 0.555.0: biblioteca de ícones.
- framer-motion 12.23.26: animações e transições.
- vaul 1.1.2: drawers/sheets acessíveis.
- sonner 2.0.7: sistema de toasts/notificações.
- next-themes 0.4.6: gestão de tema (dark/light/system).
- class-variance-authority 0.7.1: variações de classes para componentes.
- clsx 2.1.1: composição condicional de classes CSS.
- tailwind-merge 3.4.0: deduplicação/merge de classes Tailwind.

=====================================================================
5) TABELAS, DADOS NO CLIENTE E INTERAÇÃO
=====================================================================
- @tanstack/react-query 5.90.21: cache, fetching e sincronização de dados no frontend.
- @tanstack/react-table 8.21.3: motor de tabelas headless.
- recharts 3.5.1: gráficos e visualização de dados.
- date-fns 4.1.0: utilitários de datas.
- react-day-picker 9.12.0: seletor de datas.
- cmdk 1.1.1: command palette / command menu.
- @dnd-kit/core 6.3.1: base para drag-and-drop.
- @dnd-kit/modifiers 9.0.0: modificadores para comportamento de drag-and-drop.
- @dnd-kit/sortable 10.0.0: sorting/reordering via drag-and-drop.
- @dnd-kit/utilities 3.2.2: utilitários auxiliares do dnd-kit.

=====================================================================
6) VALIDAÇÃO, AUTENTICAÇÃO E SEGURANÇA
=====================================================================
- zod 4.1.13: validação e parsing de dados com schemas.
- otpauth 9.5.0: TOTP/2FA (autenticadores).
- qrcode 1.5.4 (+ @types/qrcode): geração de QR Codes para setup de 2FA.
- crypto (nativo Node.js): hashing/scrypt e operações criptográficas (sem pacote externo).

=====================================================================
7) INTEGRAÇÕES E SERVIÇOS EXTERNOS
=====================================================================
- Resend 6.9.2: envio de emails transacionais (ex.: reset de password).
- Salt Edge API v6: integração Open Banking para conexões bancárias e sincronização financeira.

=====================================================================
8) TOOLING, QUALIDADE E BUILD
=====================================================================

- ESLint 9: linting do código.
- eslint-config-next 16.0.3: regras ESLint específicas para Next.js.
- dotenv 17.2.3: carregamento de variáveis de ambiente.
- prisma CLI 7.3.0: geração de client e migrações.
